安装前须知

安装介质和形式：了解安装介质，盘、光盘、硬盘等. 目标系统：Opensuse, Debian, Ubuntu, CentOS, RHEL, Deepin, Fedora, SLES, MX Linux, Manjaro, Linux
Mint, Elementary OS, Solus, Linx, Zorin, antiX, PClinuxOS, Arch, ArcoLinux, ArchLabs, BackArch, Obarun, Artix Linux,
Puppy Linux, Tails, Slax, Kali, Mageia, Slackware, Q4OS, Archman, Gentoo, Pentoo, NixOS, Ubuntu Kylin…等，就说这么多吧~
空闲设备：一台空闲设备，或者一台有足够空间的设备。

准备工具

光盘：DVD空白光盘若干，避免刻坏。 U盘：一个或2个U盘，8G或16G以上的空U盘。大多数系统，特别是上网本和轻薄便携机，根本就不配CD/DVD-ROM驱动器，从U盘引导是安装操作系统的标准方法。
烧盘工具：UItralSO，Nero… U盘启动盘制作工具： Rufus,Ventoy,USB Image Tool,Win32DiskImager…等感觉哪个好用用那个，类似软件一大堆。

目标系统评测

关于打分：对于初学者来说，前几次的打份没有多大意义，因为你不了解这些系统，如果你只是听别人说这个系统好，你就说好这样的自我欺骗就没意义了。你要都试过在回来打分才会产出你对这个事情的观点。否则你就是个鹦鹉。 Debian, Ubuntu,
CentOS, RHEL, Deepin, Fedora, Linux Mint,Arch,Kali,Opensuse. 排名不分先后，适合你的就是好的。

系统评测标准（我自己的标准）：
    1.稳定性.     ☆☆☆☆☆
    2.社区人群广.     ☆☆☆☆☆
    3.更新速度快.     ☆☆☆☆☆
    4.目标使用功能完善.     ☆☆☆☆☆
    5.后续维护方便程度.     ☆☆☆☆☆
    6.你自己喜不喜欢.     ☆☆☆☆☆
自己打分，分又高到低排序，在开始测试系统。 
我目前用的是 Opensuse Tumbleweed kde plasma,也是自己一个一个测试过来的。

图形系统选择

关于打分：对于初学者来说，你就是个白鹦鹉，没有上色的鹦鹉，希望你可以成为一个彩色的鹦鹉，而不是单一颜色的黑鹦鹉。 gnome, KDE Plasma, xfce, Cinnamon, lxde, mate.排名不分先后，适合你的就是好的。

图形系统评测标准（我自己的标准）：
1.稳定性.     ☆☆☆☆☆
2.易用性.     ☆☆☆☆☆
3.配置复杂度.     ☆☆☆☆☆
4.自由度.     ☆☆☆☆☆
5.后期DIY指数.     ☆☆☆☆☆

自己打分，分又高到低排序，在开始测试桌面系统。 GNOME 3： An easy and elegant way to use your computer, GNOME is designed to put you in control
and get things done.

KDE Plasma： Featuring an emoji picker, a global edit mode, user feedback and much more.

Xfce:lightweight desktop environment for UNIX-like operating systems. It aims to be fast and low on system resources,
while still being visually appealing and user friendly.

Cinnamon:A Linux desktop featuring a traditional layout, built from modern technology and introducing brand new
innovative features.

Lxde:LXDE - Lightweight X11 Desktop Environment. This project aimed to provide a new desktop environment which is useful
enough and keep resource usage lower at the same time.

mate:The MATE Desktop Environment is the continuation of GNOME 2. It provides an intuitive and attractive desktop
environment using traditional metaphors for Linux and other Unix-like operating systems.
