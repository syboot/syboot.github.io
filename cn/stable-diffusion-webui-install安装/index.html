stable-diffusion-webui-inatall安装 for windows

Stable Diffusion是一款功能异常强大的AI图片生成器。它不仅支持生成图片，使用各种各样的模型来达到你想要的效果，还能训练你自己的专属模型。

WebUI使得Stable Diffusion有了一个更直观的用户界面，更适合新手用户。

一 安装CUDA and cuDNN

一.安装CUDA
1.1 下载驱动

https://developer.download.nvidia.com/compute/cuda/12.3.0/local_installers/cuda_12.3.0_545.84_windows.exe

前往下载页面 cuda_12.3.0_545.84_windows.exe

1.2添加环境变量：

1.2.1系统变量PATH中加入如下：
C:\Program Files\NVIDIA Corporation\Nsight Compute 2023.2.2\
C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.2\bin
C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.2\include
C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.2\lib
C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.2\libnvvp
C:\Program Files\NVIDIA Corporation\NVIDIA NvDLISR

1.2.2添加环境：
CUDA_PATH=C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.2
CUDA_PATH_V12_2=C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.2

1.2.3检查CUDA 10.2是否安装成功
Windows+R或者搜索栏输入cmd，打开命令提示符
nvcc -V
看到CUDA版本信息，则说明安装成功

nvcc: NVIDIA (R) Cuda compiler driver
Copyright (c) 2005-2023 NVIDIA Corporation
Built on Tue_Aug_15_22:09:35_Pacific_Daylight_Time_2023
Cuda compilation tools, release 12.2, V12.2.140
Build cuda_12.2.r12.2/compiler.33191640_0



二.配置cuDNN v8.9.0

https://developer.download.nvidia.cn/compute/cudnn/redist/cudnn/windows-x86_64/cudnn-windows-x86_64-8.9.5.30_cuda12-archive.zip

前往下载页面 cudnn-windows-x86_64-8.9.5.30_cuda12-archive.zip

2.1将之前下载的cuDNN压缩文件进行解压，复制其中的三个文件夹到C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v12.2

二 安装GIT for windows

一.安装GIT

前往下载页面 Download GIT for Windows

添加环境变量：c:\Program Files\Git\cmd

二.安装SD

2.1 install  python 3.10

前往下载页面 Download Windows Python 3.10.11 installer (64-bit)

添加PATH变量： 
D:\Program Files\python310\Scripts\
D:\Program Files\python310\

三 install Stable diffusion

一. 下载Stable diffusion + webui

git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git

cd stable-diffusion-webui
ls

二.python venv 位置

info:主要看注释部分

cat webui-user.bat



    三.python venv 使用（后期可以用到）

    cd stable-diffusion-webui
    
    run:
    .\venv\Scripts\activate
    执行完成后命令行提示符前 出现(venv) 说明已经是VENV环境了。

    退出：
    run: deactivate  
    在命令行执行deactivate命令 

    四.安装Stable diffusion + webui

    cd stable-diffusion-webui
    
    run:
    webui-user.bat

    会自动创建VENV 目录开始安装Stable diffusion + webui
    时间会很长.....

CUDA_SETUP: WARNING! libcudart.so not found in any environmental path. Searching in backup paths...

在venv  下运行
pip install bitsandbytes-windows

安装成功，重新启动SD 即可。
