音频

Mixxx

Mixxx是免费的开源软件，完全由社区驱动。 Mixxx背后没有公司-热情的DJ和程序员共同承担开发工作，他们将空闲时间专用于开发自己喜欢的DJ软件。 Mixxx将会并且永远是免费的！查看更多功能或下载Mixxx并立即开始DJ！

特点：

BPM，密钥检测和同步
BPM和音乐键检测可帮助您从库中找到完美的下一首曲目。使用主同步来匹配四首歌曲的节奏和节拍，以实现无缝混音。

DJ控制器支持
DJ控制器硬件的内置映射使您可以手动控制Mixxx的功能。使用可编程映射系统自定义您的工作流程并添加对任何MIDI或HID设备的支持。

特效
将您独特的旋转效果添加到具有音效的音轨中。通过将多个效果链接在一起并旋转所有旋钮来发挥创意。

黑胶唱片控制
使用带有时间码黑胶唱片的转盘来控制播放和刮擦数字音乐文件，就像将它们压在黑胶唱片上一样。通过切换乙烯基直通模式将音乐乙烯基混入您的设备中。

Cecilia

Cecilia是面向声音设计师的音频信号处理环境。 Cecilia使您可以使用简单的语法创建自己的GUI。 Cecilia随附了许多原始的内置模块和预设，用于音效和合成。

特点：

模块实现更多（UltimateGrainer 最先进的粒化处理：RandomAccumulator 变速记录累加器；UpDistoRes 谐振低通滤波器）
自动保存调制

Clementine

Clementine 是一款多平台音乐播放器。 它的灵感来源于 Amarok 1.4, 致力于开发一个易于使用的界面，令您能够快速地搜索和播放您的音乐。 可能是最好的平衡了易用性与灵活性的播放器。不仅限于 Linux； Clementine
在 Mac OS 和 Windows 平台上也可以获得。

特点：

搜索并播放您的媒体库
收听这些网络电台 Spotify, Grooveshark, SomaFM, Magnatune, Jamendo, SKY.fm, Digitally Imported, JAZZRADIO.com, Soundcloud, Icecast 与 Subsonic servers.
搜索播放您上传的音乐，支持 Box, Dropbox, Google Drive, 与 OneDrive
创建智能播放列表和动态播放列表。
标签式播放列表，导入或导出 M3U，XSPF，PLS 和 ASX。
支持 CUE sheet。
播放音乐CD。
可视化效果来自 projectM.
歌词以及艺人的照片与简介。
转换音频格式至 MP3, Ogg Vorbis, Ogg Speex, FLAC 或者 AAC.
编辑 MP3 和 OGG 文件的标签，管理您的音乐。
通过 MusicBrainz 获取缺失的标签。
发现，下载 Podcast 播客。
通过 Last.fm 和 Amazon 自动补全封面。
跨平台 - 完美运行于 Windows，Mac OS X 和 Linux。
原生支持 Linux (libnotify) 和 Mac OS X (Growl) 的桌面提醒。
远程控制功能，通过 安卓设备, Wii 遥控器, MPRIS 或者其它命令行。
复制您的音乐至 iPod, iPhone, MTP 或者大容量 USB 播放器。
队列管理器。

Audacity

Audacity是一种易于使用的多轨音频编辑器和记录器，适用于Windows，macOS，GNU / Linux和其他操作系统。

特点：

记录
Audacity可以通过麦克风或混音器录制现场音频，或数字化其他媒体的录制内容。

导入导出
导入，编辑和组合声音文件。以许多不同的文件格式导出录音，一次包含多个文件。

音质
支持16位，24位和32位。采样率和格式使用高质量的重采样和抖动进行转换。

外挂程式
支持LADSPA，LV2，Nyquist，VST和音频单元效果插件。奈奎斯特效果可以在文本编辑器中轻松修改-甚至您都可以编写自己的插件

编辑中
使用剪切，复制，粘贴和删除轻松进行编辑。还可以在会话中无限制地顺序撤消（和重做）以返回任意数量的步骤。

特效
LADSPA，LV2，VST和音频单元（macOS）效果的实时预览。插件管理器处理菜单中的插件安装以及效果和生成器的添加/删除。

辅助功能
可以使用键盘完全控制曲目和选择。大量的键盘快捷键。

分析
频谱图查看模式，用于可视化和选择频率。绘制频谱窗口可进行详细的频率分析。

Audacity 下载页面

特点：

播放 Ogg Vorbis、MP3、FLAC、Opus、Musepack、WavPack、WAV、AAC、MP4、音频 CD 以及 
ffmpeg 支持的所有格式（WMA、APE、MKA、TTA、SHN 等等）
libmodplug。你也可以用它来播放 Shoutcast 或者 Icecast 的音频流。

特点：

内置编解码器
定制选项
跨平台
支持每种视频文件格式
扩展可用于增加功能
能播放任何内容 - 文件、光盘、摄像头、设备及流媒体
可播放大多数格式，无需安装编解码器包 - MPEG-2, MPEG-4, H.264, MKV, WebM, WMV, MP3...
可在所有平台运行 - Windows, Linux, Mac OS X, Unix, iOS, Android ...
完全免费 - 无间谍软件，无广告，无跟踪用户的行为
添加 外观.

安装 扩展.

OGM，VIVO，ASF/WMA/WMV，QT/MOV/MP4，FLI，RM，NupleVideo，yuv4mpeg，FILM，RoQ，PVA文件。 它还可以播放VideoCD，SVCD，DVD，3ivx，RealMedia和DivX电影。

特点：

支持各种视频，音频和字幕文件

可以将所有流内容保存到文件

SMPlayer

SMPlayer 是个适用于 Windows 和 Linux 的免费媒体播放器， 内置解码器，可以播放几乎所有格式的视频和音频文件。 无需再安装任何外部解码器。 只要安装了 SMPlayer，
你就能够播放所有的格式，不用再费事查找安装解码包。

特点：

支持所有主要视频格式
内置编解码器
跨平台（Windows和Linux）
播放无广告的YouTube视频
Opensubtitles整合
UI自定义可用
基于MPlayer
SMPlayer 支持大多数熟知的格式和编码： avi、mp4、mkv、mpeg、mov、divx、h.264...内置的解码器。 您无需查找和安装第三方的解码器。
支持Youtube  可以播放 Youtube 视频，有个用于搜索 Youtube 视频的可选插件。
自带了几个皮肤和图标主题， 所以您可以轻松地更改播放器的外观。
可以从 opensubtitles.org 搜索和下载字幕。
包含很多高级功能， 如视频和音频过滤器， 更改播放速度， 音频和字幕延迟的调整，视频均衡器... 等等。
有超过 30 种语言可用， 其中包括西班牙语、德语、法语、意大利语、俄语、汉语、日语...
可用于 Windows 和 Linux。 SMPlayer 兼容 Windows XP/Vista/7/8/10。 支持二合一电脑的触摸屏幕.
是自由和开源的. 遵循 GPL 许可证。
使用屡获殊荣的 MPlayer 作为播放引擎， 它是世界上最好的播放器之一
已开始支持 mpv 。

特点：

简单的界面

支持字幕：自动加载具有匹配名称的字幕
视频显示设置（亮度，对比度）
由于使用了Solid和Phonon，因此DragonPlayer独立于任何多媒体框架或硬件抽象层
支持播放CD和DVD
