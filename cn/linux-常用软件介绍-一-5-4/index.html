电子书工具

Calibre

Calibre 是 Linux 最受欢迎的电子书应用。老实说，这不仅仅是一个简单的电子书阅读器。它是一个完整的电子书解决方案。

特点：

Calibre 通过管理元数据来排序和分组电子书。你能从各种来源下载一本电子书的元数据或创建和编辑现有的字段。
Calibre 支持所有主流电子书格式并兼容多种电子阅读器。
在转换时，你能通过改变电子书风格，创建内容表和调整边距的选项来转换任何一种电子书格式到另一种。你也能转换个人文档为电子书。

Calibre 下载页面

Okular

Okular 是另一个开源的基于 KDE 开发的跨平台文档查看器，它已经作为 KDE 应用发布的一部分了。

特点：

Okular 支持多种文档格式像 PDF、Postscript、DjVu、CHM、XPS、ePub 和其他。
支持在 PDF 文档中评论、高亮和绘制不同的形状等。
无需修改原始 PDF 文件，分别保存上述这些更改。
电子书中的文本能被提取到一个文本文件，并且有个名为 Jovie 的内置文本阅读服务。

Okular 下载页面

Bookworm

Bookworm 是另一个支持多种文件格式诸如 epub、pdf、mobi、cbr 和 cbz 的自由开源的电子阅读器。

特点：

支持epub，pdf，mobi，cbr和cbz格式的电子书。
支持编辑书籍元数据和过滤。
提供诸如放大/缩小，边距设置，线宽增加/减少之类的功能。
支持浅色，深褐色和深色的不同读取配置文件。还支持夜间模式。
为您喜欢的书上的页面添加书签，以供日后阅读。
支持全屏模式。

Bookworm 下载页面

电子邮件

Thunderbird

Mozilla Thunderbird 非常的好用的邮件客户端，虽然很多人都习惯在网页上收发邮件，不过还是有很多资深网民喜欢用本地邮件客户端收发邮件， 它是经过对 Mozilla
的邮件组件的进行重新设计后的产品，它可整合多个网络邮箱于一体，随时都能拉取邮件到本地处理，或者随时发送邮件。 简单易用，功能强大，个性化配置，Thunderbird 邮件客户端带给你全方位的体验。

特点：

Thunderbird 支持 IMAP 、POP 邮件协议
以及 HTML 邮件格式。
轻松导入您已有的邮件账号和信息。
内置 RSS 技术, 功能强大的快速搜索, 自动拼写检查等。

Thunderbird 下载页面

Claws Mail

Claws Mail对邮件和信息的管理采用MH格式，因此具有良好的通用性。此外，还可以利用fetchmail、procmail以及其它外部程序来增强它的功能。

特点：

多用户支持
线程操作
过滤器支持
SSL连接支持
X-Face功能支持
内建图片浏览功能
支持插件
邮箱的导入和导出功能
GnuPG的支持
标签支持
打印功能
排序工具
POP和SMTP协议的认证等等

Claws Mail 下载页面

KMail

KMail 是一个电子邮件客户端 。它的设计目标是希望成为一个企业级 MUA。支持文件夹、邮件过滤、浏览HTML电子邮件和国际字符集。
KMail支持OpenPGP标准，可以自动加密、解密、凭证和验证电子邮件的凭证及其经由内嵌或OpenPGP/MIME方法加密的附件。

特点：

安全-KMail具有安全的默认设置，以保护您的隐私，强大的端到端加密支持以及垃圾邮件检测。
强大-功能包括离线支持，多种发件人身份，多语言支持，强大的过滤，搜索和标记功能，邮件列表管理以及非常灵活的配置。
集成-会议邀请可以作为事件轻松添加到KOrganizer中，从KAddressBook加载地址自动完成，头像和加密首选项。
符合标准-支持标准邮​​件协议，推送电子邮件，服务器端过滤以及嵌入式OpenPGP，PGP / MIME和S / MIME。

KMail 下载页面

SeaMonkey mail

SeaMonkey 就是一个完整的 Web 应用程序套件，包括一个网页浏览器、一个高级邮件和新闻组客户端、一个IRC 聊天客户端以及一个简单的 HTML 编辑器。

特点：

SeaMonkey包含Internet浏览器
电子邮件和新闻组客户端
随附的Web Feed阅读器
HTML编辑器
IRC聊天
Web开发工具

SeaMonkey 下载页面

Evolution

使用gnome作为桌面环境的朋友一定对Evolution不会陌生，它为gnome桌面环境下电子邮件、个人信息、日程安排、日历等方面提供了一整套高效的解决方案

特点：

Evolution管理多个POP，IMAP，Microsoft Exchange和本地电子邮件帐户。
Evolution集成了日历，通讯录和电子邮件。
强大的HTML支持，包括小的HTML“模板”（例如，用于框架内嵌图像）。
灵活的过滤器和虚拟文件夹，用于消息的组织，分类和优先级。
Evolution与SpamAssassin集成以进行垃圾邮件过滤，可以轻松训练贝叶斯部分。
可以标记消息（例如用于后续操作），并可以为其分配颜色。
Evolution同时支持OpenPGP和S / MIME以进行安全消息传递。
